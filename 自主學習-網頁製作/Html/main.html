牛頓三大運動定律介紹

簡介 第一定律 第二定律 第三定律

簡介

牛頓三大運動定律由牛頓於1687年7月5日在 <自然哲學的數學原理> 中提出

其描述施加於物理的外力與物體所呈現的運動彼此間的關係

第一運動定律（慣性定律）

假若施加於某物體的外力為零，則該物體的運動速度不變 (靜止或等速運動)

牛頓第一定律表明，假若施加於某物體的外力為零，則該物體的運動速度不變。速度是向量，速度包括了運動的大小與方向。

根據定律來說 :

  • 靜止或等速度運動會保持原運動型態，直到有外力施加於物體為止
  • 等速度運動會維持其運動速度的大小和方向，直到有外力施加於物體為止

第二運動定律 (加速度定律)

外力等於受力物的質量和加速度的和

牛頓第二定律表明，施加於物體的外力等於質量與加速度的乘積。這定律又稱為「加速度定律」。

根據定律可得到一個公式 :

F = m x a

其中 F 是外力，m 是受力物的質量，a 是受力物的加速度。

依此可以得到國高中最常用的四個公式 :

  • V = Vo + a x t
  • S = (V + Vo) x t / 2
  • S = Vo x t + ½ x a x t²
  • V² = Vo² + 2 x a x S

其中 S 是位移，V 是物體當前速度，Vo 是初速，t 是時間

第三運動定律 (作用與反作用力定律)

施力於一物體時，會產生大小相等，方向相反且作用於施力者的反作用力

牛頓第三定律表明，當兩個物體交互作用時，彼此施加於對方的力，其大小相等、方向相反。

根據第三定律，力是物體與物體之間的交互作用，力必會成雙結對地出現，其中一道力稱為「作用力」，而另一道力則稱為「反作用力」。

這兩道力的大小相等、方向相反。在這兩道力之間，任何一道力都可以被稱為作用力，而其對應的力自然成為伴隨的反作用力。這成對的作用力與反作用力稱為「配對力」。

第三定律又稱為「作用與反作用定律」。
